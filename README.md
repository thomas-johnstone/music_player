# Music Player Project

## Introduction:
Simple design of the software for a mock music player application written in Java.

## Technologies:
Project is created with:
* Lorem version: 12.3
* Ipsum version: 2.33
* Ament library version: 999

## Project Status:
* v 1.0

## Setup & Launch:
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
